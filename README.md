# 🚖 OLA Data Analytics Project (Lucknow)

## 📌 Project Overview

This is an **end-to-end Data Analytics project** built using Ola ride booking data for Lucknow city.
The dataset (5,000 records) was generated using Python to simulate real-world ride scenarios and business conditions.

### 🔍 Key Highlights:

* ✅ 62% Successful bookings
* ❌ <7% Cancelled by Customers
* ❌ <18% Cancelled by Drivers
* ❌ <6% Incomplete Rides
* 📈 Higher demand & booking value observed on weekends and match days

This project demonstrates the **complete data analytics pipeline** from raw data to actionable insights.

---

## 🚀 Tools & Technologies

---

## 📈 Key Insights

* Ride success rate is ~62% with noticeable driver cancellations
* UPI & Wallet dominate as preferred payment methods
* Demand and revenue increase significantly on weekends
* Premium vehicles generate higher value but lower frequency
* Driver-related cancellations impact overall efficiency

---

## 💡 Business Recommendations

* Reduce driver cancellations through incentives & monitoring
* Optimize pricing during peak demand (weekends/events)
* Promote digital payment methods for faster transactions
* Focus on improving driver performance and ride completion rate
* Increase availability of high-demand vehicle types
